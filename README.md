我的待办清单 v1.1.0

一个轻量级、无依赖的纯前端待办清单应用，专注于简洁高效的任务管理体验。

✨ 核心功能

完整任务管理：添加、删除、编辑任务，标记完成状态

三级优先级系统：高 / 中 / 低优先级，不同颜色标识，点击快速切换

智能截止日期：

支持设置任务截止日期

当天截止显示红色高亮

未来日期显示黄色提示

已过期显示灰色标记

"下一天" 快捷按钮，点击一次日期加一天

默认无截止日期，输入框显示 "无截止日期" 提示

双主题支持：深色 / 浅色模式一键切换，自动保存用户偏好

本地数据持久化：所有数据保存在浏览器本地，刷新 / 关闭浏览器不丢失

智能排序：高优先级任务 > 截止日期近的任务 > 中优先级 > 低优先级

全响应式设计：完美适配手机、平板和电脑

流畅交互体验：

单任务编辑模式，自动保存未完成修改

点击页面空白处自动保存

支持回车键保存、ESC 键取消

任务筛选：全部 / 未完成 / 已完成

🛠️ 技术栈

HTML5：语义化页面结构

CSS3：现代样式、响应式布局、过渡动画

原生 JavaScript：无需任何框架，轻量高效

LocalStorage：本地数据存储

🚀 快速使用

在线访问

GitHub Pages：https://treatureofworld.github.io/todo-app/

下载本仓库的三个文件：index.html、style.css、script.js

用任意浏览器打开index.html即可使用
